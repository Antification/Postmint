# Postmint

Mint any X/Twitter post as an NFT on Base. Each user gets their own deterministic collection.

## Triggers
- "postmint this"
- "mint this tweet"
- "turn this post into an NFT"
- Providing an x.com/twitter.com status URL and asking to mint it.

## Usage
Install this skill to enable tweet-to-NFT minting capabilities.

## Scripts
- `scripts/mint.ts`: The primary entry point for minting requests.
